#  ADHD_Helper

A minimal, dark-mode assistive tool engineered to bypass executive dysfunction. Dump messy thought paths or overwhelming tasks and watch an integrated unauthenticated open AI engine slice it into clean, actionable, micro-step tick boxes.

## ⚡ Features
- **Planning Fatigue Defeated:** Instantly groups long text blocks into 3-5 sub-tasks.
- **Persistent Sessions:** State tracking locked instantly into browser `localStorage`. Progress survives refreshes and tab closure.
- **Audio Feedback:** Synthesized Web Audio API completion micro-ticks and sound indicators.
- **Mobile First Design:** Tailored multi-touch zones for easy task checking on mobile smartphones.

## 🛠️ Tech Stack
- Core Structure: HTML5 (Responsive Layout Engine)
- Style Skin: CSS3 (Custom Dark Palette Configuration)
- Logic Framework: Pure Vanilla ES6 JavaScript
- Model Interface: Hugging Face Inference API Gateway (`zephyr-7b-beta`)

## 📲 How to Use on Your Phone (Without An App Store)
This project is built explicitly to be loaded straight through mobile browsers using **GitHub Pages**:

1. Go to your Repository **Settings** -> **Pages**.
2. Select your `main` branch deployment layer, leave folder path as root `/`, and click **Save**.
3. Open the live deployment URL generated by GitHub on your iOS Safari or Android Chrome app.
4. Open your browser settings menu and click **"Add to Home Screen"**. 

This generates an isolated launching web shortcut right alongside your phone's native apps, offering clutter-free access anywhere, anytime, with 100% data data privacy.
